# EC-AnsibleTower

This plugin uses the Ansible Tower/AWX REST API.

It has been tested with Ansible Tower v3.8.1 and AWX 19.1.0

## Contents ##

Currently contains the following procedures

Inventories

- List Inventories
- Get Inventory
- Create Inventory

Job Templates

- List Job Templates
- Get Job Template
- Launch Job Template
- Create a Job Template

Projects

- List Projects

Organizations

- List Organizations
